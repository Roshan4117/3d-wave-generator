Three Spectrums

Electromagnetic (sin), Sound (cos), and Visual (sin·cos) — each with a distinct wave equation and colormap.

HOW IT WORKS

Flask + NumPy + Matplotlib

Each time you request a wave, the Flask backend computes a meshgrid with NumPy, evaluates the wave equation, and renders a 3D surface plot with Matplotlib — then streams it back as a base64 PNG. No static images are stored.

The REST API is fully accessible: POST /api/wave accepts JSON and returns the rendered plot plus metadata.

Electromagnetic
Z = A · sin(f · (X + Y))
Plasma colormap · default f = 1.5
Sound
Z = A · cos(f · (X + Y))
Viridis colormap · default f = 2.0
Visual
Z = A · sin(f·X) · cos(f·Y)
Inferno colormap · default f = 2.5
